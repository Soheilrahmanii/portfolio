# Soheil Rahmani — Portfolio

Personal portfolio website for Soheil Rahmani, DevOps / Platform Engineer.

## Stack

Pure HTML, CSS, and vanilla JavaScript — no frameworks, no build step.

## Features

- Animated hero section with wave SVG and teal neon profile ring
- Skills / stack showcase
- Projects section
- Experience timeline
- Contact form
- Fully responsive
- Deployed on Cloudflare Pages

## Local Development

```bash
# Any static file server works, e.g.
python3 -m http.server 8080
```

Then open `http://localhost:8080` in your browser.
